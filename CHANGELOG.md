# Drift Changelog

All notable changes to Drift will be documented here.

## [1.5.0] - 2026-07-14

### Added
- Dynamic hashtag badges in explore with trending algorithm and usage tracking
- Personalized feed ranking with engagement scoring and time decay
- GIF upload support (no compression for GIFs)
- Conversation search in messages hub
- Changelog system with in-app viewer and update notifications
- JetBrains Mono font for changelog content
- Navbar indicator for new updates with auto-dismiss on view

### Changed
- Improved post detail page layout and styling
- Enhanced feed sorting logic with user-specific scoring

### Fixed
- Facebook embed aspect ratio resolution issues
- Notification and message badges clearing each other when visiting either page

## [1.4.0] - 2026-07-10

### Added
- Direct messaging system with real-time updates
- Presence indicators (online/offline status)
- Message notifications

### Changed
- Improved mobile navigation layout
- Enhanced post card interactions

## [1.3.0] - 2026-07-05

### Added
- SOL tipping on posts
- NFT showcase in user profiles
- Leaderboard for top tippers

### Changed
- Optimized feed virtualization performance
- Improved media upload flow

## [1.2.0] - 2026-07-01

### Added
- Quote posts functionality
- Facebook and YouTube embeds
- Post analytics (views, engagement)

### Fixed
- Video upload size validation
- Media preview loading states

## [1.1.0] - 2026-06-25

### Added
- Repost functionality
- Notifications system
- User mentions in posts

### Changed
- Enhanced responsive design for mobile
- Improved avatar upload UX

## [1.0.0] - 2026-06-20

### Added
- Initial release
- Wallet-based authentication (Solana devnet)
- Posts, comments, likes
- Follow system
- User profiles
- Media uploads (images, videos)
- PWA support
